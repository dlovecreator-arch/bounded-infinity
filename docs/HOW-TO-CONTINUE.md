## Where Files Live

Everything is in: `~/bounded-infinity/`

On a Mac, you can find this by:
1. Open Finder
2. Press Cmd + Shift + G
3. Type: `~/bounded-infinity`
4. Press Enter

## How To Work With AI On This Project

### With Claude or ChatGPT
1. Tell them: "I have a project in ~/bounded-infinity. Help me with [task]."
2. They can't directly edit files, so they'll give you code to paste
3. Ask them for step-by-step instructions

### With a CLI AI Tool (like opencode)
1. Tell them: "Work in ~/bounded-infinity"
2. They can directly create and edit files
3. They can run commands on your Mac
